The Dynamic Blood Directory App is a web-based application designed to simplify the process of finding blood donors based on their blood group and location. It provides an easy-to-use interface where users can search for available donors and view their details. The application helps reduce the time required to connect blood donors with people in need, making the blood search process more efficient.

Features
Search donors by blood group.
View donor information in a simple and organized format.
Dynamic data retrieval using PHP and a database.
Responsive and user-friendly interface.
Easy navigation and fast search functionality.
Technologies Used
HTML – Used to create the structure of the web pages.
CSS – Used to design and style the user interface.
JavaScript – Used to add interactivity and improve the user experience.
PHP – Used for server-side logic and dynamic content generation.
MySQL – Used to store and manage donor information.
How It Works

The application stores donor information in a MySQL database. When a user searches for a specific blood group, PHP processes the request, retrieves matching records from the database, and displays the results dynamically. JavaScript enhances the interface by providing a smoother and more interactive experience, while HTML and CSS ensure the application is well-structured and visually appealing.

Purpose

The primary goal of this project is to create a simple and efficient blood donor directory that enables users to quickly locate blood donors when required. It demonstrates the integration of front-end and back-end web technologies to build a practical real-world application.
