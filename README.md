# 🩸 Dynamic Blood Directory App

## About

The Dynamic Blood Directory App is a web-based application that helps users quickly find blood donors based on their blood group. The system is designed to provide a simple, fast, and user-friendly way to search for donor information during emergencies. It retrieves donor details dynamically from a database, ensuring that users receive accurate and up-to-date information.

## Features

- Search donors by blood group
- Dynamic retrieval of donor information
- Responsive and user-friendly interface
- Fast and efficient search functionality
- Organized display of donor details

## Technologies Used

- HTML
- CSS
- JavaScript
- PHP
- MySQL

## Project Description

This application integrates front-end and back-end technologies to create a dynamic blood donor directory. HTML provides the structure of the website, CSS enhances its appearance, JavaScript improves user interaction, PHP handles the server-side processing, and MySQL stores and manages donor data.

When a user searches for a particular blood group, the application processes the request using PHP, fetches matching records from the MySQL database, and displays the results dynamically on the webpage.

## Purpose

The main objective of this project is to simplify the process of finding blood donors during emergencies while demonstrating the implementation of a full-stack web application using PHP and MySQL.
